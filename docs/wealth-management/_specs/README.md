## 2. Product Standards

- **Tone**: Professional, sophisticated, yet accessible for the "Modern Wealth Manager."
- **Privacy First**: All features must default to the strictest privacy (Masked) state.
- **Micro-Interactions**: All functional elements must support the "Premium Experience" (Glassmorphism, hover-states, micro-animations).
