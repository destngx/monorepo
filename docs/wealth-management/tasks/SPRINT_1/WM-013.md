**Description**:
Port the interactive `setup-oauth.ts` from the legacy Next.js app to a Go CLI. This allows developers to quickly set up their Google Sheets credentials (ID, Secret, Refresh Token) without manual `.env` editing.

- **Prerequisites**: Must live in `apps/wealth-management-engine/cmd/google_oauth_setup`.
- **Logic**: Use `golang.org/x/oauth2` and `google.golang.org/api/sheets/v4`.
  **Acceptance Criteria**:
- Prompts for `GOOGLE_CLIENT_ID` and `GOOGLE_CLIENT_SECRET` if not found in `.env.local`.
- Displays the Auth URL and prompts for the redirected callback URL.
- Automatically exchanges the code for a `GOOGLE_REFRESH_TOKEN`.
- Upserts all three variables into the workspace `.env.local`.

**Implementation Notes**:
- Replaces/enhances the existing placeholder in `apps/wealth-management-engine/cmd/google_oauth_setup/main.go`.
- Uses `bufio.Scanner` for interactive terminal prompts.
- Ensure proper file handling for `.env.local` to avoid wiping other variables.

**Development Commands**:
```bash
# Run the setup script
bunx nx run wealth-management-engine:run --args="cmd/google_oauth_setup/main.go"
```
