# Tasks

Purpose: index atomic data-contract task files.

## Rules

- one task per file
- one acceptance criterion per task
- tasks must be independently completable
- tasks must use wiki links to related docs only
- each task must reference a short phase ID from `[[../../../progress]]` (SPEC, MOCK, MVP, FULL)

## Format

- `WM-DATA-###.md`
- short title
- phase ID (SPEC, MOCK, MVP, FULL)
- phase tag
- definition of done
- verification link
