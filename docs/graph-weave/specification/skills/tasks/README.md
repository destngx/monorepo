# Tasks

Purpose: index atomic skill-contract task files.

## Rules

- one task per file
- one acceptance criterion per task
- keep items independently completable
- tasks must use wiki links to related docs only
- each task must reference a short phase ID from `[[../../../progress]]` (SPEC, MOCK, MVP, FULL)

## Format

- `GW-SKILL-###.md`
- short title
- phase ID (SPEC, MOCK, MVP, FULL)
- phase tag
- definition of done
- verification link
