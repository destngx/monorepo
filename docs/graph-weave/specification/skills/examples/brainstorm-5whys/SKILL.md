# 5 Whys: Clarify the Real Request

## Your Role in the Chain

You are **STEP 1** of 2-skill brainstorm system.

**Your job**: Clarify what user ACTUALLY needs, not what they said.

## Quick Start

1. **State stated goal** - "User said: I want to..."
2. **Ask why? 5 times** - Dig to root cause
3. **Identify real need** - Not symptom
4. **Success criteria** - What indicates success?
5. **Hand off** - Trigger skill brainstorm-first-principles next

## Output Template

```
## Stated Goal
[What user said]

## Why Chain (5 Levels)
1. Why? → [Answer]
2. Why? → [Answer]
3. Why? → [Answer]
4. Why? → [Answer]
5. Why? → [Answer]

## Real Need Identified
[Actual problem, not surface request]

## Success Criteria
[What would prove this is solved?]

## Next Step
→ Trigger brainstorm-first-principles
  to rebuild solution from first principles
```
